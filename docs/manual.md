# Manual de instruções

Antes de começar a utilizar o programa é importante ler esse manual de instruções para previnir futuros problemas e entender melhor como o programa funciona.

## Instalação

1) Caso você tenha feito a clonagem correta do repositório, e a instalação mas esqueceu de instalar os requerimentos, é possível que o programa liste que os módulos não estão instalados ou não existam.

2) Ao rodar o programa certifique-se de ler todas as opções com atenção.

3) Em caso de algum outro erro que aqui não foi listado, abra um issue na página do projeto no github e publique para que eu possa analisar e resovler corretamente.

Para garantir a instalação correta do repositório na sua máquina é necesseário estar atento ao guia de instalação presente na página do projeto.

## Funcionamento

Ao iniciar o programa de maneira correta, a primeira parte a ser carregada é o conteúdo principal que é estruturado da seguinte forma:

1) Cabeçalho: Mostra a ascii art, o nome do projeto e o do author;
2) Opções: Mostra as opções a serem escolhidas pelo usuário, onde as três primeiras fazem parte da documentação do projeto, as finais para inicialização dos programas e a última normalmente de número 00 para encerrar o programa.

Após escolher alguma função da documentação o usuário terá o tempo que desejar para ler atentamente a documentação. Quando ele iniciar o menu e o programa ele também terá o tempo necessário para escolher a opção desejada, para digitar os valores e também para analisar o gráfico.

Ao final de cada processo aparecerá uma pergunta para que o usuário escolha rodar o código novamente ou se ele quer encerrar aquele processo.

É importante ressaltar que esses programas requerem um conhecimento básico de física, envolvendo conhecimento sobre unidades de medidas, sistema internacional de unidades, termodinâmica, e matemática básica apenas para noção de unidades e reconhecimento de ângulação, números decimais e etc para melhor compreensão do usuário.
